# UA - Unsupervised Analysis

Unsupervised Learning and Clustering Analysis

## Overview

A collection of unsupervised learning experiments covering clustering, dimensionality reduction, and anomaly detection.

## Topics Covered

- K-Means Clustering
- DBSCAN density-based clustering
- Hierarchical Clustering
- PCA (Principal Component Analysis)
- t-SNE visualization

## Getting Started

```
git clone https://github.com/pawaravinash0007/UA.git
cd UA
pip install pandas numpy scikit-learn matplotlib seaborn jupyter
jupyter notebook
```
